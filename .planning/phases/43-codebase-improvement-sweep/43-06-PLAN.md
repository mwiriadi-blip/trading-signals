<action>
1. **Truncation (presentation-only)** in `dashboard_renderer/components/positions.py`:
   - Locate the loop that builds HTML rows from `trade_log`.
   - Replace iteration source with `trade_log[-200:]` — a new list slice, NOT an in-place mutation.
   - Add inline comment: `# D-06: render last 200 only; full state preserved; admin endpoint serves full log`.
   - Verify (by inspection) no code path passes `trade_log` by reference to anything that mutates it.

2. **`/admin/trades/full` route** in `web/routes/admin/__init__.py`:
   - Add route `@router.get("/admin/trades/full")` returning JSON.
   - Resolve `user_id` from the authenticated session using the existing project pattern (match how other admin routes in this file already do it). If no authenticated session → return 401.
   - Response body: `{"user_id": <id>, "trade_log": state["users"][user_id]["trade_log"]}`.
   - The handler MUST scope by the SESSION's user_id only. Do NOT accept a `user_id` query param, do NOT iterate `state["users"]`, do NOT return any other user's data.
   - Add `Content-Disposition: attachment; filename="trades-<user_id>-<date>.json"` header so it downloads.

3. **Admin dashboard link**: add a download link to `/admin/trades/full` visible only when the session is authenticated as admin. Match existing admin-only conditional render patterns.

4. **Tests in `tests/test_admin_routes.py`** (new file if absent, else append):
   - `test_admin_trades_full_anonymous_rejected`: no session → status 401 or 403.
   - `test_admin_trades_full_tenant_isolation`: seed state with users A and B each having distinct sentinel trade IDs; authenticate as A; assert response contains A's sentinels and does NOT contain ANY of B's sentinels.
   - `test_admin_trades_full_returns_all_user_rows`: seed user A with 300 trades; authenticate; assert response `len(trade_log) == 300`.

5. **Tests in `tests/test_dashboard.py`**:
   - `test_dashboard_truncates_trade_log_to_200`: seed 300 trades; render; assert HTML row count for trade-log table is exactly 200.
   - `test_dashboard_render_does_not_mutate_state`: snapshot `len(state["users"][uid]["trade_log"])` before render, render, assert length unchanged after.
</action>
