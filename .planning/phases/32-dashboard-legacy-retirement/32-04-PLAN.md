<!-- dashboard_legacy/__init__.py stub pattern (LOCKED per V-02 fix):
     ```python
     '''dashboard_legacy retired — all rendering lives in dashboard_renderer.

     Any attribute access or submodule-style import raises ImportError to catch
     accidental re-introduction. Setting __path__ = [] makes submodule imports
     (e.g. `import dashboard_legacy.render_helpers`) fall back to __getattr__,
     so they raise ImportError (not ModuleNotFoundError) with the locked message.
     '''

     __path__ = []

     def __getattr__(name):
       raise ImportError("dashboard_legacy retired — use dashboard_renderer")
     ```
     The `__path__ = []` line is LOAD-BEARING: without it Python's import machinery
     would raise ModuleNotFoundError when looking for non-existent submodule files,
     bypassing __getattr__. With __path__ = [] there are no submodule search paths,
     so `import dashboard_legacy.X` falls back to attribute access on the parent
     module, which hits __getattr__ and raises ImportError. -->
