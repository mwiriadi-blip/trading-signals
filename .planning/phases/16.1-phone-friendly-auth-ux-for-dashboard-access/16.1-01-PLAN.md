## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Public Internet → AuthMiddleware | Untrusted HTTP requests cross here; every byte (headers, query, body) hostile until proven otherwise |
| Operator's `.env` → create_app() | `WEB_AUTH_USERNAME`, `WEB_AUTH_SECRET` env vars cross from disk to in-memory at boot only |
| Operator's authenticator app → POST /verify-totp | TOTP code crosses from operator's phone (TOTP secret was previously delivered server→operator via QR) |
| auth.json on disk → auth_store.py | TOTP secret + enrollment state at rest; protected by file permissions inherited from droplet's trader user |
