### Collapsible Panel

```html
<details class="news-panel-disclosure" {open_attr}>
  <summary class="news-panel-summary">Market News</summary>
  <div class="news-panel-inner">
    <!-- optional: banner -->
    <!-- table of up to 5 headlines -->
  </div>
</details>
```

`{open_attr}` = `" open"` when `not user_prefs.get('news_panel_collapsed')` else `""`.
Collapse state field: `state['users'][uid]['news_panel_collapsed']` (bool, default `False` = open).
Source: RESEARCH.md §3 — mirrors `trace-disclosure` `<details>` pattern exactly.

### Critical-Event Banner (conditional)

Rendered only when `news_filter.has_critical_event(headlines, market_id)` returns `True`.
Appears above the headlines table, inside `.news-panel-inner`.

```html
<div class="news-banner">
  <p class="news-banner-heading">Possible market-moving news — operator review recommended</p>
</div>
```

Banner copy is locked (CONTEXT.md D-07 / ROADMAP). Do not vary this string.

### Headline Row (HTMX dismiss)

```html
<tr id="news-row-{{ title_hash }}"
    hx-post="/news/{{ market }}/dismiss/{{ title_hash }}"
    hx-target="#news-row-{{ title_hash }}"
    hx-swap="outerHTML">
  <td class="news-link">
    <a href="{{ url }}" target="_blank" rel="noopener noreferrer">{{ title }}</a>
  </td>
  <td class="news-meta">{{ publisher }} · {{ pub_date }}</td>
  <td>
    <button type="button" class="btn-news-dismiss"
            hx-post="/news/{{ market }}/dismiss/{{ title_hash }}"
            hx-target="#news-row-{{ title_hash }}"
            hx-swap="outerHTML"
            hx-trigger="click">
      Dismiss
    </button>
  </td>
</tr>
```

`rel="noopener noreferrer"` is mandatory on every headline anchor (REQUIREMENTS NEWS-01).
Jinja2 `autoescape=True` is structural — no per-field escaping needed (REQUIREMENTS NEWS-03).

### Empty State (no headlines)

Rendered inside `.news-panel-inner` when `len(headlines) == 0` after dismiss filtering:

```html
<p class="subtle">No headlines today.</p>
```

## Interaction Contract

| Interaction | Trigger | Mechanism | Server Response | Visual Outcome |
|-------------|---------|-----------|----------------|----------------|
| Dismiss headline | Click "Dismiss" button | `hx-post` on `<button>` inside `<tr>` | `POST /news/{market}/dismiss/{hash}` → `Response(content="", media_type="text/html")` | Row removed via `hx-swap="outerHTML"` on `<tr id="news-row-{hash}">` |
| Collapse panel | Click `<summary>` | Native `<details>` toggle + HTMX `hx-post` | `POST /news/panel-toggle` → updates `state['users'][uid]['news_panel_collapsed']` | Panel content collapses/expands natively; state persisted server-side |
| Panel open on load | Server render | `news_panel_collapsed` from per-user state | — | `<details open>` or `<details>` (no attribute) |

Source: CONTEXT.md D-02, D-09; RESEARCH.md §3, §4.

## Copywriting Contract

| Element | Copy |
|---------|------|
| Panel `<summary>` label | Market News |
| Critical-event banner heading | Possible market-moving news — operator review recommended |
| Dismiss button | Dismiss |
| Empty state (no visible headlines) | No headlines today. |
| Empty state (all dismissed) | No headlines today. |
| Panel collapse toggle aria-label | Toggle Market News panel |
| Headline link title attribute | (none — link text is the headline title itself) |

Notes:
- "Possible market-moving news — operator review recommended" is locked copy from CONTEXT.md D-07 / ROADMAP. Do not rephrase.
- No error state copy is needed for the dismiss interaction: the HTMX empty-200 pattern silently removes the row. If the network fails, HTMX leaves the row in place (no error shown — acceptable for a dismiss action on a supplementary panel).
- No destructive confirmation: dismiss is reversible daily (auto-expire on date change per D-08). Standard `.btn-row` confirm pattern is not required.

## Accessibility Contract

| Element | Requirement | Implementation |
|---------|-------------|----------------|
| `<details>/<summary>` | `aria-expanded` auto-synced | Existing JS in `assets.py::_TRACE_TOGGLE_JS` already handles all `<details>` on `DOMContentLoaded` — news panel gets this for free (RESEARCH.md §3) |
| Headline links | Opens in new tab | `target="_blank" rel="noopener noreferrer"` mandatory |
| Dismiss button | Touch target ≥ 44px | `min-height: var(--touch-target-min)` (44px) |
| Critical-event banner | Role: advisory | No `role="alert"` — banner is not urgent enough for screen-reader interrupt; rendered as static content |
| Panel summary | Focus-visible | Inherited from `summary:focus-visible` rule in Phase 25 CSS |
