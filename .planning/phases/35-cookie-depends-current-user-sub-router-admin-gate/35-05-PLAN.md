## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| browser→/admin/* | Untrusted client requests cross into admin-gated routes |
| middleware→dependency→handler | Three-layer auth chain: signed cookie → request.state.user_id → role check |
| route registration order | Future admin routes must register on admin_router (not application) — invariant test enforces |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-35-05-01 | Elevation of privilege | admin gate bypass via wrong-router registration | mitigate | TestAdminRouteInvariant walks app.routes after create_app() and fails the build if any `/admin/*` path lacks require_admin in its dependency chain. This is the milestone-wide regression guard for Phase 36+ admin routes. |
| T-35-05-02 | Spoofing | session cookie forgery | mitigate | itsdangerous URLSafeTimedSerializer signature on every decode (existing); tampered uid → BadSignature → falls through; tampered role field cannot exist because role is read from auth.json live, not from cookie. |
| T-35-05-03 | Elevation of privilege | header-auth bypass to admin | mitigate (intentional) | D-06: header auth sets user_id=None; require_admin raises 403. TestAdminGate403Sweep::test_header_auth_returns_403 is the regression guard. |
| T-35-05-04 | Tampering | backward-compat shim abuse | mitigate | Shim is read-only — calls get_user_by_email(payload['u']) and consults auth.json live. Cannot be used to escalate: even with a forged legacy cookie (valid signature, payload['u']='attacker'), the shim returns None for any user not in auth.json. The bootstrap-gap test proves this. |
| T-35-05-05 | Spoofing | tsi_trusted reaching /admin/* | mitigate (acceptable) | Per Pitfall 1 + A1, tsi_trusted leaves user_id=None in Phase 35; admin using a trusted device sees 403 on /admin/ping and must re-login via TOTP. Acceptable for grace period; revisit in Phase 36 if friction. |
| T-35-05-06 | Denial of service | startup invariant test cost | accept | Walks app.routes once per test invocation — O(routes) which is bounded (≤ ~50 routes today). Test runtime ~10 ms. |
| T-35-05-07 | Repudiation | admin route access logging | accept | Existing AuthMiddleware logging unchanged; admin route hits are logged via the standard request log. Detailed audit log for admin actions is a Phase 37+ concern. |
| T-35-05-08 | Information disclosure | legacy cookie shim leaking row contents | mitigate | The shim resolves a uid and sets request.state.user_id; the row dict from get_user_by_email is NOT stored on the request or echoed in any response. Only the uid string propagates. |
</threat_model>
