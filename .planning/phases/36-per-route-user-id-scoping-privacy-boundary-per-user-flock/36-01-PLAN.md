  def mutate_state(
    mutator: Callable[[dict], None],
    path: Path = Path(STATE_FILE),
  ) -> dict:
    fd = os.open(str(path), os.O_RDWR | os.O_CREAT, 0o600)
    try:
      fcntl.flock(fd, fcntl.LOCK_EX)
      try:
        state = load_state(path=path, _under_lock=True)
        mutator(state)
        io._save_state_unlocked(state, path=path)
        return state
      finally:
        fcntl.flock(fd, fcntl.LOCK_UN)
    finally:
      os.close(fd)

<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Cookie → current_user_id() | Session cookie uid is decoded in web/dependencies.py; routes trust the decoded uid without re-verifying ownership |
| uid → state['users'][uid] | Route passes uid from cookie into state layer; a forged or replayed cookie could access another user's bucket |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-36-01 | Elevation of Privilege (IDOR) | state_manager/__init__.py::mutate_user_state | mitigate | uid sourced exclusively from Depends(current_user_id) which validates the session cookie signature; lock file keyed on uid prevents cross-user write races |
| T-36-02 | Information Disclosure | GET /admin/users | mitigate | FastAPI response_model=list[PublicUserSummary] strips all non-listed fields at serialization; paper_trades/equity/entry_price cannot appear in response body |
| T-36-03 | Tampering | state/users/{uid}.lock | accept | lock files contain no state data; an attacker creating a stale lock file causes DoS not data corruption; the inner state.json flock preserves correctness regardless |
| T-36-04 | Elevation of Privilege | PATCH /admin/users/{uid}/disable | mitigate | route is behind require_admin Depends; non-admin 403 enforced by admin sub-router |
</threat_model>
