<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| user_id (cookie) → _apply closure | uid from session cookie selects which user bucket _apply mutates; an expired/replayed cookie with another user's uid = IDOR |
| /admin/users JSON response | FastAPI response_model strips fields; if response_model is absent or wrong, trade data leaks |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-36-05 | Elevation of Privilege (IDOR) | paper_trades/_apply + trades/_apply | mitigate | _apply navigates state['users'][user_id] where user_id = Depends(current_user_id); a user can only mutate their own bucket; entity not found in their bucket raises 404 naturally |
| T-36-06 | Information Disclosure | GET /admin/users | mitigate | response_model=list[PublicUserSummary] enforced; FastAPI JSON serialization strips all non-model fields including paper_trades, equity_history, trade_log |
| T-36-07 | Tampering | record_trade uid parameter | mitigate | uid flows from Depends(current_user_id) at route level into _apply closure; cannot be supplied by client request body |
| T-36-08 | Elevation of Privilege | PATCH /admin/users/{uid}/disable | mitigate | behind require_admin Depends on the admin sub-router; non-admin gets 403 before handler is called |
</threat_model>
